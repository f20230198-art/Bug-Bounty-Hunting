# 🐛 Bug Bounty Hunting Workshop

> **A 3-day hands-on workshop that takes you from zero to finding real security vulnerabilities.**

---

## 🎯 What Is This?

This is a **3-day practical workshop** on bug bounty hunting — the art of finding security vulnerabilities in websites and web applications. Companies like Google, Facebook, and Microsoft **pay real money** to hackers who responsibly report bugs. This workshop teaches you how.

```
Day 1 → The Hacker Mindset      (Recon, SQL Injection, XSS)
Day 2 → Server-Side Attacks     (Path Traversal, Command Injection, SSRF, XXE, File Upload)
Day 3 → Real World Hunting      (IDOR, JWT Attacks, Deserialization, WordPress, Bug Bounty Platforms)
```

---

## 🧑‍💻 Who Is This For?

- **Complete beginners** — no prior cybersecurity knowledge required
- **CS/IT students** curious about security
- **Developers** who want to understand how attackers think
- Anyone who thinks hacking is cool 😎

## ⚠️ Important: Legal Disclaimer

> **Never test for vulnerabilities on systems you don't have explicit permission to test.**
> Bug bounty programs give you legal permission — always read the program's scope and rules before testing.
> Unauthorized hacking is illegal and unethical.
